# Mouse pseudogenes

A character vector containing all known pseudogenes in the mouse genome
as of may 10th 2024.

## Usage

``` r
pseudogenes_mouse
```

## Format

### pseudogenes_mouse

A character vector with 3663 elements.

## Source

<https://rna.sysu.edu.cn/dreamBase2/scrna.php?SClade=mammal&SOrganism=mm10&SDataId=0&SProteinID=0>
