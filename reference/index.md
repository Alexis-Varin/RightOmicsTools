# Package index

## Seurat

### Visualization

- [`Barplot_Cell_Proportion()`](https://alexis-varin.github.io/RightOmicsTools/reference/Barplot_Cell_Proportion.md)
  : Bar plot of the cell proportions in each identity
- [`DotPlot_Heatmap()`](https://alexis-varin.github.io/RightOmicsTools/reference/DotPlot_Heatmap.md)
  : Dot plot or heatmap of average gene expression in each identity
- [`Cell_Heatmap()`](https://alexis-varin.github.io/RightOmicsTools/reference/Cell_Heatmap.md)
  : Heatmap of gene expression in each cell
- [`Grid_VlnPlot()`](https://alexis-varin.github.io/RightOmicsTools/reference/Grid_VlnPlot.md)
  : Square stacked violin plot of gene expression in each identity

### Helper Functions

- [`Right_DietSeurat()`](https://alexis-varin.github.io/RightOmicsTools/reference/Right_DietSeurat.md)
  : Reduce the size of a Seurat object
- [`GSEA_Signatures()`](https://alexis-varin.github.io/RightOmicsTools/reference/GSEA_Signatures.md)
  : Gene signatures from GSEA-MSigDB pathways
- [`Find_Annotation_Markers()`](https://alexis-varin.github.io/RightOmicsTools/reference/Find_Annotation_Markers.md)
  : Get the top markers for fast annotation

  smoother curves along pseudotime
- [`heatmapSmoothers()`](https://alexis-varin.github.io/RightOmicsTools/reference/heatmapSmoothers.md)
